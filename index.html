Migración a SQLite Exitosa

El sistema integral ha sido migrado con éxito a una arquitectura cliente-servidor con una base de datos SQLite externa.

La base de datos original (411 estudiantes, 55 docentes y 41 asistentes) se encuentra almacenada en campanario.db.

Para iniciar la aplicación, ejecuta el servidor backend desde tu terminal:

python server.py Abrir Aplicación en 127.0.0.1:8080
